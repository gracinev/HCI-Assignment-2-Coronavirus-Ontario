## Description
Coronavirus Ontario is an iOS application that aims to provide statistics and education regarding Coronavirus in Ontario.
The application allows for users to check on the daily statistics regarding Coronavirus in both Ontario and in Canada. Users have the ability to see up-to-date statistics of total cases, active cases, resolved cases, and total Coronavirus deaths. Focusing on education, the app contains valuable information regarding what Coronavirus is and how to prevent the spread of the virus. Finally, the application allows for the user to chat with a volunteer about COVID-19 concerns, as well as to call a volunteer toll free.
